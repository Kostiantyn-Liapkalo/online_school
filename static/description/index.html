Online School

Complete Learning Management System for Odoo

Overview

Online School is a comprehensive learning management system built for Odoo 19.0. It enables organizations to create and manage online courses, enroll students, track progress, and issue certificates.

Key Features

📚 Courses

Create structured courses with lessons, difficulty levels, and scheduling.

👨‍🎓 Students

Student registration with contact integration and enrollment tracking.

👨‍🏫 Teachers

Assign teachers to courses with specialization and qualification tracking.

📝 Lessons

Multi-format lessons: video, text, quizzes, and assignments.

📊 Progress Tracking

Monitor student progress with percentage completion and grades.

🏆 Certificates

Generate completion certificates with unique numbering.

Models

Model Description
Courses Course management with lessons, difficulty levels, and scheduling
Students Student registration linked to partners with enrollment tracking
Teachers Teacher management with specializations and qualifications
Enrollments Student-course relationships with progress tracking
Lessons Course content with multiple formats (video, text, quiz)

Security & Access Control

  • School Administrator: Full access to all features
  • School Teacher: Manage courses and students, create content
  • School Student: View courses and track own progress
